🔧 Cara Menjalankan

CV ini bisa di-deploy gratis menggunakan GitHub Pages:

Masuk ke repo > Settings > Pages.

Pilih branch main > /root > Save.

CV akan tampil di URL:

https://username.github.io/cv-nezya/
